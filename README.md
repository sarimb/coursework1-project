# Airline Passenger Forecasting Project

This project uses Meta’s Prophet model in R to analyse and forecast airline passenger demand.

## Overview

* Time series analysis of monthly airline passengers (1949–1960)
* Identification of trend and seasonality patterns
* Forecasting future passenger demand using Prophet

## Tools Used

* R
* RStudio
* Prophet
* R Markdown
